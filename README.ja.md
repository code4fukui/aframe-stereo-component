# aframe-stereo-component

A-Frameの立体視機能を提供するコンポーネントです。

## デモ
以下にデモページをご用意しています。
- [立体視ビデオ](http://oscarmarinmiro.github.io/aframe-stereo-component/basic_video/index.html)
- [立体視画像](http://oscarmarinmiro.github.io/aframe-stereo-component/basic_image/index.html)
- [立体視立方体](http://oscarmarinmiro.github.io/aframe-stereo-component/two_boxes/index.html)

## 機能
- 立体視カメラコンポーネント: モノスコピック表示時にカメラに表示する左右の目を指定できます。
- 立体視コンポーネント: 立体視表示するオブジェクトの左右の目に表示するかを指定できます。立体視ビデオ再生にも対応しています。

## 使い方
`aframe-stereo-component.js.min.js`をインクルードした上で、カメラや立体視表示するオブジェクトに対してコンポーネントを設定します。  
詳しい使い方は[README.md](https://github.com/oscarmarinmiro/aframe-stereo-component/blob/master/README.md)をご覧ください。
